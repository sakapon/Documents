# 経営で MVV と呼ばれているものの出典
経営の世界で「Mission, Vision and Values (MVV)」と言われている用語があります。  
日本企業の Web サイトの会社概要や企業理念のページを閲覧すると、
- ミッション
- ビジョン
- バリュー

と、まるで定型のように項目が並んでいるのをよく見かけると思いますが、これのことです。  
この記事では、MVV と呼ばれている経営テンプレートの出典を確認していきます。
