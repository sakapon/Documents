# 数論・素数に関する機能を実装する (C#)
機能は次の通りです。

### Euclid 互除法
- 最大公約数 (GCD)
- 最小公倍数 (LCM)
- Euclid 互除法の拡張
  - ax + by = 1 の解

### 素数
- 素因数分解
  - O(√n)
- 約数の列挙
  - O(√n)
- 素数判定
  - O(√n)
- n 以下の素数の列挙
  - およそ O(n)
- m 以上 M 以下の素数の列挙
  - およそ O(max(√M, M - m))
